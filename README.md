# chatNode

## 准备工作

  1. 先在本地安装好mongodb客户端，可以装个Robo 3T的数据库视图工具
  2. 还需要安装redis服务器，我们数据session缓存存放在redis服务器上,视图工具叫rdm，这些都自行查找文档安装即可

## usage

> *首先需要下载项目：git clone git@github.com:Jiangjinjin1/chat_node.git*

> `cd chat_node`

> `npm i` 下载依赖项

> 本地起服务执行 `npm run dev`

> 服务启动后在浏览器输入 `localhost:9000` 即可
